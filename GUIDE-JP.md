## 🚀 **DÉMARRAGE EXPRESS** (2 minutes)

### **Option A: Play.bat (RECOMMANDÉ)** ⭐

1. **Ouvre** GitHub: https://github.com/Freecey/neon-protocol
2. **Clique** sur **`Code`** → **`Download ZIP`**
3. **Dézippe** le fichier dans `C:\Games\neon-protocol`
4. **Double-clique** sur **`play.bat`**
5. ** attends** que le navigateur s'ouvre
6. **JOUE !** 🎮

---

### **Option B: Installer Node.js (si play.bat ne marchepa)**

**Si play.bat ne marche pas:**

1. **Rend-toi ici:** https://nodejs.org/
2. **Clique** sur **"LTS Version"** (le gros bouton vert)
3. **Installe** comme n'importe quel programme Windows
4. **Redémarre** ton PC
5. **Clique** sur **`play.bat`**
6. **JOUE !**

---

## 🎮 **COMMENT JOUER**

Quand le navigateur s'ouvre:
1. **Clique** sur **"Commencer"**
2. Utilise **← →** pour bouger
3. **Espace** pour sauter
4. **Évite** les ennemis
5. **Collecte** les pièces
6. **Survive** 60 secondes!
